1.5-Mile Run — test protocol
Measures aerobic endurance. A single continuous timed run over 1.5 miles (2,414 m) on a flat, measured course. Standing start, single all-out effort, time from start signal to finish line crossing.
Equipment required
  • Stopwatch or electronic timing — accurate to 0.1s
  • Measured 1.5-mile course, 400m track (6 laps), or 1/4-mile track
  • GPS device with ±1% distance accuracy (alternative to measured course)
Test guidelines
  • Standing start at the start line
  • Single all-out continuous effort — not paced for training
  • Time from start signal to finish line crossing
  • Course grade must not exceed ±2% over any 400m segment
  • No external pacing (vehicle or bicycle escort)
Valid surfaces
  • Certified running track
  • Measured flat road or path
  • GPS-verified route (±1% tolerance)
Treadmill is not valid for certified administration. Belt speed and grade vary across machines, and treadmill running energy cost differs materially from overground.
